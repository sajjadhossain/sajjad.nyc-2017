# twoPointO

## Start Locally

1. Download [NodeJS](https://nodejs.org/en/download/)
2. Download [NPM](https://www.npmjs.com/package/npm)
3. `npm install`
4. `npm run start`

## Test

1. `npm install webdriverio@v3.2.5 -g`
2. `npm install selenium-standalone -g`
3. `selenium-standalone install`
4. `wdio wdio.conf.js`
